## 🏗️ Technical Architecture

This repository is not just a static site—it's a demonstration of modern web engineering principles. It features a custom **glassmorphic design system**, heavily optimized scroll behaviors, and a sleek bento-grid layout.

### Tech Stack
- **Framework:** React 19 + TypeScript + Vite
- **Routing:** TanStack Router (File-based routing, ultra-fast prefetching)
- **Styling:** Tailwind CSS v4 (Using the latest `--theme()` syntax and cascading layers)
- **Animations:** Framer Motion (Orchestrating micro-interactions, spring physics, and scroll-linked animations)
- **Design:** Custom HSL "Aurora" color palette

### Key Features
- **Dynamic Hero Section:** Ambient drifting background blobs and subtle noise overlays.
- **Bento Grid Layouts:** Re-engineered project card stacking with solid backdrops to prevent visual bleed-through.
- **Advanced State Management:** Clean separation of concerns with a centralized data hub for projects and experience.
